# Personal Portfolio Website

## Project Summary
This is my personal portfolio website, designed and developed to showcase my identity, skills, and projects. The website is fully responsive and built using **HTML**, **CSS**, and **Tailwind CSS (configured using Tailwind CLI)**.  
It demonstrates my abilities in **UI/UX design** and **frontend development**.

## Features Implemented
- Multi-page responsive design (Home, About, Portfolio, Contact).  
- Hero section with personal introduction and tech skills icons.  
- Professional Summary and Education sections.  
- Projects section with interactive cards linking to GitHub projects.  
- Tools & Skills section with icons.  
- Portfolio buttons linking to Behance, GitHub, and LinkedIn.  
- Contact form with input fields for Name, Email, and Message.  
- Downloadable CV button.  
- Social media links in header and contact section.  
- Tailwind CSS for styling and hover/transition effects.
